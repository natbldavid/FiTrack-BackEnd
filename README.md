# FiTrack

FiTrack is a personal fitness tracking platform for managing:

- Food tracking
- Exercise and gym sessions
- Body weight progress
- Activity logging
- Nutrition goals

## Backend

ASP.NET Core Web API  
Entity Framework Core  
SQL Server  

## Features

- JWT authentication
- Food logging
- Workout planning
- Live gym session tracking
- Today dashboard
- Activity tracking

## Project Structure

- Controllers
- Services
- DTOs
- Entities
- EF Configurations

## Status

Backend API completed.
Frontend in development.
